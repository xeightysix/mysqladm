
## Installation
Make Script Executable
```sh
chmod +x mysqladm.sh
```
Edit Config File
```sh
Replace MYSQLPASSWD='none' with you MySQL ROOT password
```
Run
```sh
./mysqladm.sh
```

## List Of Commands [ Made Easy ]

 - List All Databases
 - List All Users
 - [Create | Drop] Databases
 - [Add | Remove ] Users
 - Dump Database To File
 - Show GRANTS For A Specific User
 - GRANT ALL Privileges

## Dump Files
Dump Files Location
```sh
./dumps/
```
